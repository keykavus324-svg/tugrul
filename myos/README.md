# MyOS - Linux Tabanlı İşletim Sistemi

Bu proje, sıfırdan geliştirilen bir Linux tabanlı işletim sistemidir.

## Proje Yapısı

```
myos/
├── boot/        # Bootloader ve boot ile ilgili kodlar
├── kernel/      # Çekirdek kodları
├── drivers/     # Donanım sürücüleri
├── include/     # Başlık dosyaları (header files)
├── user/        # Kullanıcı alanı programları
├── tools/       # Derleme ve araç scriptleri
└── docs/        # Dokümantasyon
```

## Geliştirme Aşamaları

1. ✅ Proje yapısı oluşturuldu
2. ⏳ Bootloader kurulumu
3. ⏳ Kernel başlangıç kodu
4. ⏳ Bellek yönetimi
5. ⏳ Kesme (interrupt) yönetimi
6. ⏳ Donanım sürücüleri
7. ⏳ Kullanıcı alanı

## Gereksinimler

- NASM (Netwide Assembler)
- GCC Cross Compiler (i686 veya x86_64)
- QEMU (test için)
- GRUB veya benzeri bootloader
